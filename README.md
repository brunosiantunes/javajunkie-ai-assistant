# JavaJunkie - AI Assistant ☕🤖

> **Assistente IA para auxiliar nos meus estudos em Java, feito em Python.**

### 🔗 [Acesse o JavaJunkie ao vivo aqui!](https://brun-javajunkie.streamlit.app/)

---

## 📖 Sobre o Projeto

Este projeto é uma aplicação web interativa que utiliza um LLM (Large Language Model) via API da Groq para atuar como um tutor sênior em Java. O objetivo principal é auxiliar nos estudos de **Programação Orientada a Objetos (POO)**, ecossistema da **JVM** e boas práticas de desenvolvimento backend.

A infraestrutura e a interface gráfica foram desenvolvidas em **Python** utilizando o framework **Streamlit**, provando a capacidade de integrar diferentes ecossistemas tecnológicos.

## 🛠️ Tecnologias Utilizadas

- **Python 3**: Motor da aplicação.
- **Streamlit**: Framework para a interface web responsiva.
- **Groq API**: Processamento de linguagem natural e lógica da IA.
- **Git/GitHub**: Controle de versão e CI/CD via Streamlit Cloud.

---

## 🚀 Como Usar - 🌐 Versão Web

1. Acesse o assistente através do link: [brun-javajunkie.streamlit.app](https://brun-javajunkie.streamlit.app/).
2. Obtenha sua **API Key** gratuita no site da [Groq Cloud](https://console.groq.com/keys).
3. Na barra lateral esquerda do JavaJunkie, insira sua chave no campo indicado.
4. Digite sua dúvida sobre Java, POO ou arquitetura backend no chat e aguarde a resposta do assistente.

---

## 🌟 Créditos e Inspiração

Este projeto é uma adaptação e evolução de um estudo de caso prático proposto no curso de **Fundamentos de Linguagem Python da Data Science Academy (DSA)**. O código original foi modificado e recalibrado (via _System Prompt_) para focar estritamente no ecossistema Java.
