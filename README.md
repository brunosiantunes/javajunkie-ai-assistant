# JavaJunkie - AI Assistant ☕🤖

Assistente IA para auxiliar nos meus estudos em Java, feito em Python.

## Sobre o Projeto

Este projeto é uma aplicação web interativa que utiliza um LLM (Large Language Model) via API da Groq para atuar como um tutor sênior em Java. O objetivo principal é auxiliar nos estudos de Programação Orientada a Objetos (POO), ecossistema da JVM e boas práticas de desenvolvimento backend.

A infraestrutura e a interface gráfica foram desenvolvidas em **Python** utilizando o framework **Streamlit**, provando a capacidade de integrar diferentes ecossistemas tecnológicos.

## Créditos e Inspiração

Este projeto é uma adaptação e evolução de um estudo de caso prático proposto no curso gratuito de **Fundamentos de Linguagem Python da Data Science Academy (DSA)**. O código original foi modificado e recalibrado (via System Prompt) para focar estritamente no ecossistema Java.

## Tecnologias Utilizadas

- **Python 3** (Motor da aplicação)
- **Streamlit** (Interface Web)
- **Groq API** (Comunicação com o modelo de IA)
